# MPC Image Source

MPC Image Source is a free and open-source DirectShow filter. The filter allows to open graphics images in video players. Recommended MPC-BE 1.5.7.6180 or newer.

## Minimum system requirements

* An SSE2-capable CPU
* Windows 7 or newer
